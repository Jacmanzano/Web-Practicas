## 📚 Detalle de las Prácticas

### 🔹 [T0 - Introducción](./T0%20-%20Introduccion)
Primeros pasos en el desarrollo web.
* Configuración del entorno de desarrollo (Visual Studio Code).
* Estructura básica de un proyecto web.
* Creación de la primera página "Hola Mundo".

### 🔹 [T1 - Elementos HTML5](./T1%20-%20Elementos%20HTML5)
Creación del esqueleto de la web utilizando marcado semántico.
* Uso de etiquetas de texto (`h1-h6`, `p`, `span`).
* Enlaces (`a`) absolutos y relativos.
* Inserción de contenido multimedia (`img`).
* Listas ordenadas y desordenadas.

### 🔹 [T2 - Introducción CSS](./T2%20-%20Introducción%20CSS)
Dar estilo y diseño visual a la estructura HTML.
* **P1 - CSS Básico:** Vinculación de hojas de estilo externas.
* **P2 - Selectores:** Jerarquía, bordes y propiedades de fondo.
* **P3 - Tipografía y Listas:** Uso de fuentes personalizadas y viñetas con imágenes.
* **P4 - Cajas y Z-Index:** Posicionamiento avanzado de capas y superposición de elementos.

### 🔹 [T3 - JavaScript](./T3%20-%20JavaScript)
Dotar de interactividad y lógica a la web.
* **P1 - Asistente Virtual:** Chatbot básico con condicionales `if/else` e interacción `prompt/alert`.
* **P2 - Operadores:** Calculadora lógica y comparaciones matemáticas.
* **P3 - Manipulación de Datos:**
    * **Arrays:** Algoritmos de ordenación.
    * **Strings:** Manejo de cadenas de texto.
    * **Cola (FIFO):** Estructura de datos "Primero en entrar, primero en salir".
